# master/

Approved master mascot art and dated reference iterations.

## Files

- `pawstreak-mascot-master.png` — the single source of truth. Includes the
  full Bailey + Meiomi character turnaround, expressions, adventure poses,
  app icon studies, sticker pack, share-card poses, color palette, and style
  notes. **Treat this as the brand bible.**
- `pawstreak-mascot-YYYY-MM-DD-HHMM-SS.png` — historical reference renders
  kept alongside the master so we can trace how the look evolved.

## What goes here

- Approved versions of the brand sheet
- Promoted iterations that should outlive a single experiment
- Reference dumps from art reviews

## What does **not** go here

- Loose ideation / scrap renders (keep those out of the repo)
- Cropped per-pose exports (those belong in their dedicated subfolder, e.g.
  `stickers/`, `share/`)
