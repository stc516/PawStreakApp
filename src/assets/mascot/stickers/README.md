# stickers/

Reaction stickers and emote-style mascot moments. These show up at micro
delight points — adventure complete, streak save, pack progress, share modal.

The master sheet ships an initial sticker pack. Use those names exactly when
exporting:

| Sticker        | Mascot  | Vibe                           |
| -------------- | ------- | ------------------------------ |
| Let's go!      | Bailey  | Hype to start                  |
| Good day!      | Bailey  | Warm cheerful celebration      |
| Pawsome!       | Meiomi  | Gentle approving praise        |
| Need coffee    | Bailey  | Self-aware, low-energy humor   |
| So tired       | Bailey  | Earned post-adventure exhaustion |
| Booty wiggle   | Meiomi  | Pure joy                       |
| You got this!  | Meiomi  | Quiet encouragement            |
| Adventure buds | Duo     | The two of them together       |

## Naming

```
bailey-lets-go.png
bailey-need-coffee.png
meiomi-pawsome.png
duo-adventure-buds.png
```

## Sizing

Square aspect ratio, transparent background. Export at multiple sizes
(128 / 256 / 512) so the sticker reads at any in-app surface.
