# share/

Social share-card artwork. The hero illustration that anchors the share modal
after an adventure, and any branded social images we render server-side or
export from the app.

## Master sheet inspiration

The brand sheet ships three approved share-card poses:

1. **Bailey on the beach** — *"Explored tide pools · +45 Adventure XP · Shore Dog vibes"*
2. **Meiomi sunset patrol** — *"Sunset mission with my human · +30 Adventure XP · Pathfinder Energy"*
3. **Bailey reflecting** — *"Another great day. Another memory for the story."*

These should feel like the user's dog *just had* the adventure — never
generic, never stocky.

## Naming

```
share-bailey-tide-pools.png
share-meiomi-sunset.png
share-bailey-reflect.png
```

## Specs

- 1200 × 1500 (4:5) for portrait social shares
- 1200 × 630 (1.91:1) for OG / link-preview cards
- Always include the PawStreak wordmark in the corner — see master sheet
