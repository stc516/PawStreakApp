# PawStreak: App Screens Overview

This document provides a comprehensive overview of the PawStreak mobile app design system, including core user flows, discovery features, and the emotional onboarding experience.

## Brand Identity: Cinematic Amber
- **Backgrounds:** True black (#0A0A0A) for a premium, cinematic feel.
- **Accent:** Warm amber/orange (#FF9500) glow accents for emotional warmth.
- **Typography:** DM Sans (or similar) clean sans-serif hierarchy.
- **Layout:** Floating dark cards with generous spacing and rounded corners.
