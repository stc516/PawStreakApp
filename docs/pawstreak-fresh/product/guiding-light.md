# Guiding Light

> The single sentence we optimize for when tradeoffs appear.

---

## North star

**Make today's walk feel like it was written for this dog — then make sure they never forget it.**

Everything else — streaks, path tiers, auth, analytics — exists to deepen that moment, not compete with it.

---

## Decision filter

When evaluating any feature, UI, or refactor, ask:

1. Does it strengthen the **Today → Plan → Walk → Remember** loop?
2. Does it feel like a **keepsake**, not a utility dashboard?
3. Is the dog the **protagonist** in copy and visuals?
4. Can we ship it **without** a new route or backend?
5. Would a tired owner at 6pm understand it in **3 seconds**?

If two or more answers are "no," defer or cut.
